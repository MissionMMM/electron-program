## 日志

此框架由 **Electron** 、 **Vue3** 、**Ant-design-vue** 组成，由于 Ant-design-vue 全局引入的影响，组件包会稍显臃肿。

未来计划是用与开发个人工作台，主要功能是调色、文字转换、代码加解密展示、图片格式转换等。

多数计划尚在考察阶段，暂时计划所有功能以纯前端的形式实现。

**需要先下载依赖**

```
npm install
```

**然后可以打开开发模式(默认热加载)**

```
npm run electron-dev
```

**并可以构建安装包**

```
npm run dist
```

已经解决所有问题并能完美运行。后续开发持续推进中

## 文件说明

1、npmrc 文件为nsis的镜像文件，请勿删除和做任何处理

2、src/main.js 中有配置路由的公共跳转方法，无需另设，如有细节变动直接在公共代码里调整即可
